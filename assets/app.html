Providers

一个或多个 LLM,按 failover 顺序自动降级。密钥建议用 $ref:env:NAME 走环境变量注入。

Failover 顺序

前面的 provider 挂了自动切到后面。至少填一个。

记忆与 Embedder

SkillStore 路径、MemoryStore 数据库、语义召回用的 embedder。默认 hash 能跑但召回质量差。

外部渠道绑定

接到微信 / 飞书 / 企微 / Telegram / 钉钉。微信扫码登录后,直接在卡片里「启动收发」,不用开第二个终端。

绑定一次 · 重启保留 — 微信 token 缓存在 data/wechat_token.json,飞书 / 企微 / TG / 钉钉的 app_id / app_secret 存在 config.toml,重启 bonsai serve 自动复用。勾选「开机自启」后连启动收发都免了。

Agent 参数

控制每轮最多工具调用、prompt 预算。token 超软阈值时启动梯度压缩,超硬阈值强行弹出老消息。

日志

日志级别 + prompt cache 命中率统计的落盘位置。

定期清理

长期跑下,bot 进程内会起一个 daemon 线程周期清理老的 session 和 evidence 日志,不依赖外部 cron。可以关,也可以调保留期 / 频率。 此段不可删,只能改值。
